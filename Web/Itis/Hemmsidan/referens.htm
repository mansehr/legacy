1 Textattribut
font-family Anger typsnitt/typsnittsfamilj Det är säkrast att ange fler än ett typsnitt. Om det första inte finns installerat i besökarens dator visas nästa o.s.v.

Det är smart att avsluta med en typsnittsfamilj. Familjerna: sans-serif, serif, cursive, monospace och fantasy. NN 4+ stöder inte cursive och fantasy.
font-style Anger om texten ska vara normal eller kursiv. Värden:
normal, italic (kursiv) eller oblique (också kursiv).

{font-style: italic}
font-variant normal, small-caps

{font-variant: normal}

 Dåligt stöd för detta attribut.
font-weight Fet eller normal text. Värden: normal eller bold.

{font-weight: bold}

 Det går också att skriva någon av siffrorna 100, 200, … 900. Exempel: {font-weight: 900}. 900 är fetast och 100 är minst fett. IE 4+ stöder också värdena bolder och lighter.
font-size Anger textens storlek i pt, px, cm, mm, em, i procent eller med beteckningar.

{font-size: 45px}
{font-size: 200%}

 Värdena kan också ges med (absoluta) beteckningar: xx-small, x-small, small, medium, large, x-large, xx-large. Det går också att ange de relativa beteckningarna larger och smaller.
font Snabbmetod för att ange en rad värden samtidigt.

P {font: bold italic small-caps 19pt/180% arial, verdana }

 Ordningen på värdena ska vara: font {font-weight font-style font-variant  font-size/line-height font-family}. Alla värden behöver inte anges.
letter-spacing Bestämmer avståndet mellan tecknen

{letter-spacing: 2px }

 Stöds ej av NN 4+. Däremot av IE 4+, och Mozilla 13
line-height Anger vertikalt avstånd mellan textrader.

{line-height: 13px}
text-decoration
 Värdena är underline, line-through, overline, blink, none

{text-decoration: underline}

 IE 4 och 5 stöder inte "blink" och NN 4+ stöder inte "overline".
text-align Anger om texten ska vara vänster- eller högerställd, centrerad eller marginaljusterad. Värden: left, right, center, justify

{text-align: right}

 

 Om webbläsaren inte stöder "justify" ersätts det normalt med "left".
text-indent Ger texten ett indrag

{text-indent: 30px }

 Det kan vara effektfullt att ange negativa värden.
text-transform Används för att omvandla texten till versaler

{text-transform: capitalize}

Med text-transform kan texten omvandlas till versaler, bara små bokstäver eller att varje ords första bokstav blir en versal. Värdena är "capitalize", "uppercase", "lowercase" och "none".
word-spacing Bestämmer avståndet mellan ord

{word-spacing: 3px }

 Dåligt stöd för detta attribut
2 Färg- och bakgrundsattribut
color Textens färg anges med hexadecimala tal, färgnamn eller RGB-notation.

{color: red}
{color: #FFAACC}
{color: #FAC}
{color: rgb(23, 56, 188)}
{color: rgb(100%, 12%, 0%)}

 RGB-värden
RGB står för red, green och blue. Värden mellan 0-255 kan anges.Först anges värdet för rött, sedan grönt och sist blått. Det går också att använda procenttal.
background-color Anger ett blockelements bakgrundsfärg med hexadecimala tal, färgnamn eller RGB-notation.

{background-color: #03A22F}

 De flesta Background-attributen fungerar dåligt eller inte alls i NN 4+.

 
background-image Ger ett HTML-element en bakgrundsbild:

{background-image: url(bakgrund.gif) }
background
-repeat		 Anger hur bakgrundsbilden ska repeteras. Värden: repeat, no-repeat, repeat-x, repeat-y

{background-repeat: no-repeat; background-image: url(bakgrund.gif) }
background-
attachment		 Anger om bakgrundsbilden ska ligga fast eller "följa med" när besökaren skrollar upp eller ned på sidan. Värden: scroll, fixed.

{background-attachment: fixed; background-image: url(bakgrundsbild.gif) }

 Fungerar endast med <BODY>
background-
position		 Anger bakgrundsbildens position. Värden: top, bottom, left, center, right eller siffervärden.

{background-position: top right; background-image: url(bakgrund.gif) }

 
background Snabbmetod för att samtidigt ange värden för alla bakgrundsattribut:

{background: url(bakgrund.gif) #FAFAFA no-repeat bottom left }

 
3 Blockattribut
float Styr över om ett blockelement "flyter" till vänster eller höger. Värdena är left, right och none.

P {width: 300px; float: right}


 Effekten av "float: right" är i det här fallet att det 300 pixlar breda block som definieras av <P> "flyter" till höger och alltså visas på höger sida av skärmen istället för som standard till vänster.
width Anger ett blockelements bredd.

{width: 30%}
height Anger ett blockelements höjd.

{height: 700px}
margin-top Anger den övre marginalens storlek.

{margin-top: 225px}

 Marginalangivelser används för att formatera blockelement, alltså HTML-koder som automatiskt skapar radbrytningar före och efter. Exempel är <DIV>, <P> och <H2>.
margin-bottom Nedre marginalen

{margin-bottom: 1cm}
margin-right Högra marginalen

{margin-right: 55mm}
margin-left Vänstra marginalen

{margin-left: 120px}
margin Alla fyra marginalerna på en gång.

{margin: 35px}

 

 Alla fyra marginaler kan också ges olika värden genom att skriva dem en efter en:
{margin: 2px 5px 1px 3px}
I ordningen övre marginal - höger - nedre - vänster.
padding Padding anger hur mycket luft det ska vara mellan innehållet i ett blockelement och blockets ram/kantlinje.

{padding: 15px}

 Man kan ange värden för "padding-left", "padding-right" o.s.v. på samma sätt som för "margin" ovan.

 
border-top
-width		 Anger övre ramens tjocklek. Värden: thin, medium, thick eller siffervärden.

{border-top-width: 10px}
border-right
-width		 Anger högra ramens tjocklek.

{border-right-width: 10px}
border-bottom
-width		 Anger undre ramens tjocklek.

{border-bottom-width: 10px}
border-left
-width		 Anger vänstra ramens tjocklek.

{border-left-width: 10px}
border-width Anger alla fyra ramarnas storlek. Värden: thin, medium, thick eller siffervärden.

{border-width: 10px}
{border-width: 10px 23px 21px thin}
border-color Bestämmer ramens färg:

{border-color: #003355}
border-style Anger typ av ram. Värden: none, solid, dotted, dashed, double, inset, outset, groove och ridge.

{border-style: ridge}
border-top Bestämmer övre ramens tjocklek.

{border-top: 2px double gray}

 

 Först anges ramens tjocklek. Därefter ramens typ och sist färgen.
border-right Anger högra ramens tjocklek.

{border-right: 5px inset green}

 
border-bottom Anger nedre ramens tjocklek.

{border-bottom: 2px solid red}
border-left Anger vänstra ramens tjocklek.

{border-left: 2px ridge black}
border Snabbmetod för att samtidigt ange ramens tjocklek, typ och färg.

P {border: 1px solid green}
4 Klassificeringsattribut
display Används normalt för att dölja objekt på en sida. Värden: block, inline, list-item

.dold {display: none;}

 Andra värden är block, inline, list-item
list-style-type Anger hur de små "punkterna" eller siffrorna i listor ska se ut.

li {list-style-type: circle}

ul {list-style-type: square}

Möjliga värden: disc, circle, square, decimal, lower-roman, upper-roman, lower-alpha, upper-alpha, none.
5 Innehållsplacering
position Anger positionen för ett element. Värden: absolute och relative. Position + värde måste kombineras med något av eller bägge attributen left och top.

{position: absolute; left: 100px; top: 312px}

 

 Använd position inom en DIV-kod, annars blir det problem i NN.
left Anger hur långt ifrån vänsterkanten ett element ska börja visas. Om elementet x är ett underelement, alltså är placerat inom ett annat element y och y är absolut placerat, börjar x visas med utgångspunkt från y:s vänsterkant. X "ärver" y:s positionering.

Värden: siffervärden, procent.

{position: absolute; left: 200px; top: 10%}

 

 Kan bara användas tillsammans med "position: absolute" eller "position: relative".
top Anger hur långt från den övre kanten ett element ska börja visas. Övrigt: samma princip som för attributet left (se ovan).

Värden: siffervärden, procent.

{position: absolute; top: 30px; left: -35px}		 Kan bara användas tillsammans med "position: absolute" eller "position: relative".
z-index Bestämmer ordningen i djupled. Om två element är placerade på samma ställe kan man med z-index bestämma vilket som ska synas. Ett element med högre z-index-värde täcker ett med lägre.

Värden: 1,2,3... ¥

{position: absolute; top: 30%; z-index: 343}

 
visibility Anger om ett element ska synas eller inte.

Värden: visible, hidden, inherit

{visibility: inherit}

 

 Om värdet inherit används ärver det aktuella elementet "synbarhetsegenskapen" från ett annat element (huvudelementet) som det är placerat inom.
6 Pseudoklasser
A:link

A:visited

A:hover

A:active

 

 

 

 A:link {text-decoration: none; color: blue}

A:visited {color: gray}

A:hover {text-decoration: underline; background-color: white; color: blue}

A:active { color: lime }


Upp till sidans början

 A:hover fungerar inte i NN 4+, men däremot i alfaversionen av NN 5. "Hover" talar om hur länken ska se ut när muspekaren rörs över länken.

Pseudoklasserna måste anges i samma ordning som till höger. Om A:hover skrivs före A:visited visas inte A:hover.
